# ADempiere390 JapanEdition
ADempiere390LTSをベースにした日本商習慣対応バージョンです。

日本でのADempiere導入時に得られた日本企業の商習慣やニーズ、  
各業界の業務の実情に詳しい方の情報をもとにして、  
コミュニティのADempiere390LTSバージョンを改良したバージョンを作成していきます。
***
ADempiere390 Japan Edition. based on ADempiere 390 LTS.  
Sorry, this repository is for Japanese only.  

but we will feed this project achievement back to the official ADempiere community  
by adempierejapan/adempiere repository forked from official community.

# 真にビジネスで実用化されるオープンソースERPを目指して

真にビジネスで実用化されるオープンソースERPを目指すには、  
まだまだ、日本固有の要件に対するカスタマイズが必要です。

特に、各業界特有の業務のやり方にあったカスタマイズが必要で、  
それには、ソフトウェア開発技術に詳しい方は、もちろん、  
開発技術を有していなくても、各業界特有の業務の実情に詳しい方からの情報が必要です。  

実現してほしい機能やカスタマイズしてほしい内容がありましたら、私たちに連絡してください。  
そして、このリポジトリのIssueページに、是非投稿してください。  
みなさんの力が真にビジネスで実用化されるオープンソースERPの実現につながります。  

そして、みなさんの情報とお力をもとにカスタマイズされたオープンソースERPが完成されれば、  
さまざまな業界・業種で桁違いの基幹システム導入が可能になるでしょう。  

インターネット上で、見ず知らずの人たちが力をあわせ、  
それぞれが培った知識や経験を共有し、  
さらに、それぞれの技術をもって、社会に大きく貢献するソフトウェアを完成させる。  
これこそ、オープンソースプロジェクトがなせる力だと思います。  

このリポジトリのWikiにあるインストール手順や各機能説明により、ADempiereについての詳細を知ることができます。  
あなたも是非、このプロジェクトに参加してください。
